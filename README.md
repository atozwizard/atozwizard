## 대표 프로젝트
- `company_analyzer`: 기업 요구와 사용자 경험을 같은 분석 언어로 연결하는 멀티에이전트 구조

- `poketdogam`: 사용자 경험과 서비스 전달 구조를 전면에 둔 제품형 프로젝트
